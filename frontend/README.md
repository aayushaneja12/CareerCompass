# CareerCompass  
**AI-Powered Career Readiness Assistant**

This project is developed as part of the **Bachelor of Data Science Capstone Project II** at **SP Jain School of Global Management**.

CareerCompass is an AI-driven system designed to support students in making informed career decisions through **career path recommendations, skill gap analysis, and personalized guidance**.

---

# Overview

Students today have access to many career-related tools, but these tools are often:

- fragmented  
- not personalized  
- lacking clear direction  

CareerCompass addresses this by providing a **structured and intelligent workflow** that helps users:

- identify suitable career paths  
- understand their current skill level  
- recognize gaps in their readiness  
- receive actionable next steps  

Instead of answering isolated questions, the system guides users through a **complete career decision journey**.

---

#  Key Features

##  Career Path Recommendation (Implemented)

Suggests suitable career roles based on:

- user interests  
- strengths  
- skill indicators  

Includes reasoning behind recommendations.

---

##  Skill Gap Analysis (Implemented)

Compares:

- current user skills  
- expected skills for a target role  

Outputs:

- missing skills  
- weak areas  
- prioritized improvement areas  

---

## Personalized Suggestions (Partially Implemented)

Provides guidance such as:

- what skills to learn  
- learning order  
- project recommendations  
- career preparation strategies  

---

## Resume Tailoring (Prototype)

Aligns resumes with job descriptions by:

- identifying missing keywords  
- improving content relevance  
- suggesting restructuring
